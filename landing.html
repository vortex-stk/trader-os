FAQ

Perguntas frequentes

O Trader OS funciona com qualquer prop firm?▾
Sim. As principais (FTMO, Topstep, The5%ers, etc.) já vêm pré-configuradas com as regras corretas. Para qualquer outra firma, você pode configurar os limites manualmente.
Meus dados ficam seguros?▾
Sim. Usamos Supabase com Row Level Security — cada usuário só acessa os próprios dados. Screenshots ficam no Supabase Storage com URLs privadas. Nunca compartilhamos dados de trading com terceiros.
Posso importar meu histórico do MT4/MT5?▾
Sim, no plano Pro. Basta exportar o relatório de Account History do MetaTrader como CSV e fazer upload no Trader OS. O sistema detecta o formato automaticamente.
A análise de IA é realmente útil?▾
Sim. Com pelo menos 10 dias de dados, a IA (Claude) identifica em quais dias da semana você performa melhor, quais setups têm melhor RR, padrões emocionais e recomendações específicas para os próximos 7 dias.
Funciona no celular?▾
Sim. O Trader OS é um PWA — pode ser instalado no iPhone e Android como um app. O layout é totalmente responsivo para registro rápido de trades mobile.
Posso cancelar a qualquer momento?▾
Sim, sem taxa de cancelamento ou período mínimo. Cancele pelo portal do cliente Stripe em qualquer momento. Seus dados ficam preservados por 30 dias após o cancelamento.
